# image_to_static_vec

This creates rs file that defines static Vec<u8> by lazy_static for mono image data.

# Usage

```
cargo run "./tmp/nums.json" > "./tmp/vec.rs"
```

## Command

```json
{
  "img_path": "./tmp/nums.png",
  "array_length": 50,
  "prefix": "vec_num",
  "height": 10,
  "mode": "bool",
  "x_list": [
    3,
    ..
  ],
  "labels": [
    "1",
    ..
  ]
}
```
